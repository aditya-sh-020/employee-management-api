# Employee Management System

A Spring Boot REST API project for managing employees.

## Technologies

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features

- Add Employee
- Get All Employees
- Get Employee By ID
- Update Employee
- Delete Employee

## API Endpoints

POST /api/employees

GET /api/employees

GET /api/employees/{id}

PUT /api/employees/{id}

DELETE /api/employees/{id}
